## Short description (до 200 символов)

## Full description

MonitorTune — служебная утилита для Windows, обеспечивающая управление параметрами яркости и контрастности подключённых внешних мониторов через системный значок в области уведомлений. Программа предназначена для пользователей, использующих один или несколько внешних мониторов в составе рабочей конфигурации.

Управление параметрами реализовано посредством стандартного интерфейса DDC/CI (Display Data Channel / Command Interface), встроенного в большинство современных мониторов и передаваемого по сигнальным кабелям совместно с видеосигналом. При поддержке монитором стандарта DDC/CI (что характерно для моделей, выпущенных с 2015 года) программа выполняет управление параметрами без использования аппаратных кнопок устройства. На отдельных моделях стандарт DDC/CI требует предварительной активации в экранном меню (OSD) монитора.

Возможности:
— Раздельное управление яркостью и контрастностью каждого подключённого монитора
— Доступ через значок в области уведомлений без открытия дополнительных окон
— Одновременная поддержка нескольких внешних мониторов с независимыми элементами управления
— Совместимость с интерфейсами HDMI, DisplayPort, DVI и USB-C (DP Alt Mode)
— Контроль частоты обращений к монитору и отсутствие фонового опроса в целях обеспечения стабильности контроллера устройства
— Работа без прав администратора

Принцип работы: программа использует стандартизованные коды VCP (Virtual Control Panel) — 0x10 для управления яркостью и 0x12 для управления контрастностью, аналогичные применяемым встроенным экранным меню монитора. Модификация драйверов либо системных настроек Windows не осуществляется.

Совместимость: поддерживается работа с внешними мониторами, реализующими стандарт DDC/CI и подключёнными по интерфейсам DisplayPort, HDMI, DVI или USB-C. Встроенные дисплеи ноутбуков не поддерживаются — управление их параметрами реализуется штатными средствами Windows.

Конфиденциальность: программа не выполняет сбор персональных данных, не устанавливает сетевых соединений, не использует учётные записи и не реализует механизмы телеметрии. Все параметры конфигурации сохраняются исключительно в локальном хранилище пользователя.

Системные требования: Windows 10 версии 1809 и выше, Windows 11. Поддерживаемые архитектуры процессоров — x64 и ARM64. Язык интерфейса: русский.

Поддержка: вопросы и сообщения о проблемах принимаются через страницу приложения в Microsoft Store.
