# MoniTune — описание для Microsoft Store

Тексты для Partner Center → Store listing.

## Short description (до 200 символов)

MoniTune — служебная утилита для управления яркостью и контрастностью внешних мониторов и яркостью встроенных дисплеев ноутбуков через значок в области уведомлений Windows. Используются стандарты DDC/CI и WMI.

## Full description

MoniTune — служебная утилита для Windows, обеспечивающая управление параметрами яркости и контрастности подключённых внешних мониторов через системный значок в области уведомлений, а также яркостью встроенных дисплеев ноутбуков. Программа предназначена для пользователей, использующих один или несколько мониторов в составе рабочей конфигурации.

Управление параметрами внешних мониторов реализовано посредством стандартного интерфейса DDC/CI (Display Data Channel / Command Interface), встроенного в большинство современных мониторов и передаваемого по сигнальным кабелям совместно с видеосигналом. При поддержке монитором стандарта DDC/CI (что характерно для моделей, выпущенных с 2015 года) программа выполняет управление параметрами без использования аппаратных кнопок устройства. На отдельных моделях стандарт DDC/CI требует предварительной активации в экранном меню (OSD) монитора.

Управление яркостью встроенных дисплеев ноутбуков (eDP) реализовано через WMI-интерфейс Windows (`WmiMonitorBrightness` / `WmiMonitorBrightnessMethods`). Управление контрастностью встроенных дисплеев в стандарте WMI не предусмотрено.

Возможности:
— Раздельное управление яркостью и контрастностью каждого подключённого внешнего монитора, а также яркостью встроенного дисплея ноутбука
— Синхронный режим: одно перемещение регулятора применяется ко всем мониторам одновременно
— Связка «яркость↔контраст» внутри одного монитора
— Ночной режим с плавным переходом и автоматическим восстановлением дневных значений
— Доступ через значок в области уведомлений без открытия дополнительных окон
— Автозапуск при входе в Windows (по команде пользователя из меню)
— Автоматическая проверка обновлений на публичном репозитории GitHub с криптографической проверкой подписи (Ed25519 + Authenticode); проверку можно отключить в настройках
— Одновременная поддержка нескольких внешних мониторов и eDP с независимыми элементами управления
— Совместимость с интерфейсами HDMI, DisplayPort, DVI и USB-C (DP Alt Mode)
— Контроль частоты обращений к монитору и отсутствие фонового опроса в целях обеспечения стабильности контроллера устройства

Принцип работы: программа использует стандартизованные коды VCP (Virtual Control Panel) — 0x10 для управления яркостью и 0x12 для управления контрастностью — для внешних мониторов, и WMI-методы `WmiSetBrightness` — для встроенных. Модификация драйверов либо системных настроек Windows не осуществляется.

Совместимость: поддерживается работа с внешними мониторами, реализующими стандарт DDC/CI и подключёнными по интерфейсам DisplayPort, HDMI, DVI или USB-C, а также с встроенными дисплеями ноутбуков (eDP).

Конфиденциальность: программа не выполняет сбор персональных данных, не использует учётные записи и не реализует механизмы телеметрии. Единственное сетевое соединение — периодическая проверка новых версий на публичном репозитории GitHub (примерно раз в 4 часа); проверку можно отключить в окне «Настройки». При аварийном завершении локально сохраняется диагностический отчёт в папке LocalCache\crashes без отправки. Все параметры конфигурации сохраняются в локальном хранилище пользователя.

Системные требования: Windows 10 версии 2004 (20H1, build 19041) и выше, Windows 11. Поддерживаемая архитектура — x64. Язык интерфейса: русский.

Поддержка: вопросы и сообщения о проблемах принимаются через страницу приложения в Microsoft Store.

- Иконки уже сгенерированы (Assets/Square*, Wide*, StoreLogo, SplashScreen).
- Скриншоты приложения (минимум 1, рекомендуется 4–9) — окно с регуляторами и контекстное меню значка.
