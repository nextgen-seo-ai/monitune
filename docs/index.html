01 · Протокол

Реальный DDC/CI, не оверлей

Команды идут по dxva2.dll напрямую в EDID-канал монитора. Регулируется backlight, а не альфа-канал поверх экрана — цвета остаются правильными.

02 · Мульти-монитор

Синхронно на всех подключённых

Один ползунок — все мониторы. Или каждый отдельно. Связка яркость↔контраст, автоопределение GPU (Nvidia/AMD/Intel) и подстройка задержек под транспорт (HDMI, DP, USB-C, TB4).

03 · Ночной режим

Расписание с плавным переходом

Автоматический тёплый режим по времени суток. Горячие клавиши на регулировку. Автостарт через StartupTask API. Иконка в трее не мешает.

Подпись и целостность

Дистрибуция только через GitHub Releases: TLS + иммутабельные теги + аудит. Подмена файла между релизом и вами невозможна без компрометации самого GitHub.

01
Manifest подписан Ed25519
Публичный ключ верификации зашит в приложение compile-time. Приватный ключ хранится только в CI secret и оффлайн-бэкапе.
02
SHA-256 из подписанного манифеста
Приложение сверяет хэш скачанного MSIX с полем в manifest.json. Manifest без валидной подписи отвергается до чтения содержимого.
03
Authenticode + WinVerifyTrust
До установки WinVerifyTrust проверяет цепочку подписи MSIX и сверяет thumbprint издателя с зашитой константой.
04
Downgrade protection
Установка версии меньше текущей запрещена. Обход только через явный флаг в настройках — защищает от отката на уязвимую сборку.
