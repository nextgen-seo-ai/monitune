MoniTune — установка на другом компьютере
==============================================

Что это:
  Утилита для управления яркостью и контрастностью внешних мониторов
  через DDC/CI. Иконка в области уведомлений Windows.

Системные требования:
  Windows 10 версии 1809 или новее, или Windows 11.
  Архитектура: x64.

УСТАНОВКА:
  1. Скопируйте всю эту папку на целевой компьютер.
  2. Дважды кликните по install.bat (потребует прав администратора —
     нужны для установки сертификата подписи в системное хранилище).
  3. Дождитесь сообщения "Установка завершена".
  4. Иконка-солнышко появится в области уведомлений.
     При первом запуске Windows может скрыть её в "переполненный лоток"
     (стрелка вверх рядом с часами) — можно перетащить на основной лоток.

УДАЛЕНИЕ:
  Запустите uninstall.bat
  ИЛИ Параметры -> Приложения -> найти MoniTune -> Удалить.

ЧТО ВНУТРИ ПАПКИ:
  install.bat                          — скрипт установки
  uninstall.bat                        — скрипт удаления
  MonitorTune_1.0.0.0_x64.msixbundle   — пакет приложения
  MonitorTune.cer                      — публичный сертификат подписи
  WindowsAppRuntime-x64.exe            — Windows App SDK Runtime (зависимость)

ВОЗМОЖНЫЕ ПРОБЛЕМЫ:

* "Не удалось установить из-за недоверенного сертификата"
  Скрипт install.bat импортирует сертификат автоматически.
  Если что-то пошло не так, импортируйте вручную:
    PowerShell от имени администратора:
    Import-Certificate -FilePath MonitorTune.cer ^
                       -CertStoreLocation Cert:\LocalMachine\Root
    Import-Certificate -FilePath MonitorTune.cer ^
                       -CertStoreLocation Cert:\LocalMachine\TrustedPeople

* "Windows App Runtime отсутствует"
  Запустите вручную WindowsAppRuntime-x64.exe и подождите.

* Утилита не видит монитор
  Включите DDC/CI в меню (OSD) монитора. У разных производителей опция
  называется по-разному: DDC/CI, DDC, Auto-Source-Switch и т.п.

* Иконка в трее не появилась
  Windows 11 по умолчанию скрывает иконки новых приложений в
  "переполненный лоток". Нажмите стрелку вверх возле часов и перетащите
  значок MoniTune на основной лоток.
